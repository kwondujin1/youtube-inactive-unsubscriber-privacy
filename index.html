Privacy Policy

Last updated: July 8, 2026

YouTube Inactive Unsubscriber is a Chrome extension that helps users identify and unsubscribe from inactive YouTube channels.

Data Accessed

The extension accesses YouTube data needed to provide its single purpose:

  • The user's YouTube subscription list.
  • Channel titles, channel IDs, thumbnails, and latest upload dates.
  • Data needed to unsubscribe from channels selected and confirmed by the user.

The extension uses the YouTube login session already present in the user's Chrome profile. It does not require a separate Google OAuth connection.

Data Stored

The extension stores only local settings in Chrome storage, such as the month threshold and whether to include channels with no videos.

Scan results are displayed locally in the extension page. CSV export is generated locally by the browser.

Data Sharing

The extension does not sell, rent, share, or transfer user data to third parties.

The extension does not use an external backend server.

Data Transfer

The extension sends requests only to YouTube domains needed for the requested feature, including https://www.youtube.com/.

These requests are made from the user's browser using the user's existing YouTube login session.

Account Changes

If the user chooses to unsubscribe, the extension asks for confirmation and then sends a YouTube request to remove the selected subscription. This action changes the user's YouTube account.

Revoking Access

Users can remove the extension from Chrome at any time. Users can also sign out of YouTube in Chrome to prevent the extension from accessing the logged-in YouTube session.

Contact

For privacy questions, contact the developer through the Chrome Web Store support channel for this extension.
